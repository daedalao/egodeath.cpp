# egodeath.cpp

A C++ implementation of the egodeath agent/system.

## Features
- TUI with dashboard, reasoning display, and chat history.
- System monitoring (CPU, RAM, GPU).
- Command history and progress indicators.
- Memory management.

## Build Requirements
- CMake
- C++17 or later
- ncurses
- fmt library

## Building
```bash
mkdir build
cd build
cmake ..
make
```

## Running
```bash
./src/egodeath
```
